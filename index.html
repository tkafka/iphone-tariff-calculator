Calculation Method:
Retail Price = (BOM Cost × (1 + Tariff Rate) + Other Costs) × (1 + Profit Margin)

Where:
  • BOM Cost: Cost of components (as % of base price, estimated to 55 % for 16 Pro)
  • Tariff Rate: U.S. import tax
  • Other Costs: Calculated as % of the initial Base Price (adjustable, default 13%)
  • Intended Profit Margin: Adjustable (39% is current iPhone Margin)

Example: If Base Price = $999, BOM % = 55%, Tariff = 25%, Other Costs % = 13%, Profit Margin = 39%,
Then BOM Cost = $999 × 0.55 = $549.45
Other Costs = $999 × 0.13 = $129.87
Retail = ($549.45 × (1 + 0.25) + $129.87) × (1 + 0.39) ≈ $1135.19, rounded up to $1199
